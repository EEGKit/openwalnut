Usage

The point connector takes point data and sends out fiber (track) data.
To connect points you have to add them to the same fiber via shift-clicking in the 3D view on them.
Different fiber can be selected in the properties tab.
You can also shift-drag in the 3D view to select multiple points at once.

Shortcuts

Shortcut Description
strg + z Undos the last action
strg + y Redos the last action
shift + a Adds a new track
shift + t Toggles the visibility of the track
(shift) + x Doubles the scale of the x axis (pressing shift halves it)
(shift) + y Doubles the scale of the y axis (pressing shift halves it)
(shift) + z Doubles the scale of the z axis (pressing shift halves it)
shift + c Confirms the current prediction
